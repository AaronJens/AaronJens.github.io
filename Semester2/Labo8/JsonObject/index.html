Welk formaat wordt er eigenlijk gebruikt om het Date object?

ISO 8601-formaat, vb: 2025-04-03T12:00:00.000Z

Wat gebeurt er als eenzelfde object meermaals voorkomt bij het opbouwen van de JSON tekst?

eenzelfde object meerdere keren voorkomt bij het genereren van een JSON-tekst, wordt het object telkens volledig opnieuw opgenomen. JSON ondersteunt geen referenties of gedeelde objecten, waardoor dit kan leiden tot redundantie en een grotere bestandsgrootte.
